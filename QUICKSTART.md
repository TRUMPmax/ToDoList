# 快速启动指南

## 1. 安装依赖

```bash
pip install -r requirements.txt
```

## 2. 运行应用

```bash
python app.py
```

应用将在 `http://localhost:5000` 启动

## 3. 使用说明

### 添加任务
1. 点击右上角"添加任务"按钮
2. 填写任务信息（标题必填）
3. 选择优先级和分类
4. 点击"创建"

### 管理任务
- **完成任务**: 点击任务卡片上的"完成"按钮
- **编辑任务**: 点击"编辑"按钮修改任务信息
- **删除任务**: 点击"删除"按钮（需确认）
- **排序任务**: 直接拖拽任务卡片进行排序

### 查看数据分析
1. 点击"数据分析"按钮
2. 查看各种统计图表：
   - 每周完成任务数（柱状图）
   - 任务类别分布（饼图）
   - 专注时间趋势（折线图）

### 使用AI推荐
1. 点击"AI推荐"按钮
2. 查看系统推荐的专注时长
3. 点击"开始专注"使用计时器
4. 选择要专注的任务
5. 系统会根据推荐时长自动设置计时器

### 专注计时器
- 点击"开始"开始计时
- 点击"暂停"暂停计时
- 点击"停止"结束并记录专注时间

## 4. 功能特性

### 任务过滤
- 按状态过滤：全部/待完成/已完成
- 按优先级过滤：全部/高/中/低
- 按分类过滤：全部/工作/学习/一般/创意

### 优先级标记
- 🔴 高优先级：红色边框
- 🟠 中优先级：橙色边框
- 🟢 低优先级：绿色边框

### 数据可视化
- 实时更新的图表
- 多维度数据分析
- 直观的统计信息

## 5. 注意事项

1. **首次使用**: 系统需要一些历史数据才能提供准确的AI推荐，建议先创建一些任务并记录专注时间
2. **模型训练**: 当专注时间记录达到10的倍数时，系统会自动训练模型
3. **数据持久化**: 所有数据存储在 `database.db` 文件中
4. **定时爬取**: 系统会在每天凌晨2点自动爬取数据（需要应用运行）

## 6. 故障排除

### 端口被占用
如果5000端口被占用，可以修改 `app.py` 最后一行的端口号：
```python
app.run(debug=True, host='0.0.0.0', port=5000)  # 改为其他端口
```

### 数据库错误
如果遇到数据库错误，可以删除 `database.db` 文件，系统会自动重新创建。

### 依赖安装失败
确保使用Python 3.8+，并尝试：
```bash
pip install --upgrade pip
pip install -r requirements.txt
```

## 7. 开发模式

应用默认运行在调试模式，代码修改会自动重载。

生产环境建议使用：
```bash
pip install gunicorn
gunicorn -w 4 -b 0.0.0.0:5000 app:app
```
